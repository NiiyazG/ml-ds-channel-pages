# ML. Урок 47/4 — Сбор данных
## Парсинг и веб-скрапинг

Когда данных нет в открытых датасетах и API — парсим сайты. Собираем текст, таблицы, цены, отзывы.

**Плюсы:** можно собрать данные, которых больше нигде нет.

**Минусы:** сайты меняют вёрстку, блокируют ботов, а ещё это юридически серая зона.

### Инструменты
- BeautifulSoup, Scrapy (Python) — для статики
- Selenium, Playwright — если сайт на JS
- ParseHub, Octoparse — визуальные парсеры без кода

### Практический совет
Сначала проверь robots.txt сайта — там написано, что можно парсить. Добавь user-agent, задержки между запросами (time.sleep(1-3)). Не долби сайт — уважай чужой сервер.

### Типичная ошибка
Запустил парсер в цикле без паузы. Сайт заблокировал IP через 3 минуты. Теперь данные собраны только на 10% от запланированного. Ставь задержки, прокси и логируй каждый шаг.
