## Урок 28: Пропущенные значения (Missing Values)

**Главная мысль:**
Данные с пропусками — это нормально. Важно уметь их правильно обрабатывать.

### Что такое пропущенные значения?
Это пустые ячейки, null, NaN, None — там, где данные не собраны или потеряны.

Примеры:
- возраст клиента не указан
- цена товара отсутствует
- дата рождения unknown

### Почему они появляются:
1. Данные не были собраны
2. Ошибки ввода
3. Технические сбои
4. Поле не применимо

### Как находить:
```python
import pandas as pd

df.isnull().sum()  # считаем пропуски по колонкам
df.isnull().mean()  # % пропусков
```

### Стратегии обработки:

**1. Удалить строки/колонки**
```python
df.dropna()  # удалить все строки с пропусками
df.dropna(axis=1, thresh=1000)  # удалить колонки с >1000 пропусков
```
Используй, если пропусков мало (<5%).

**2. Заполнить константой**
```python
df['age'].fillna(0)
df['city'].fillna('Unknown')
```

**3. Заполнить статистикой**
```python
df['age'].fillna(df['age'].mean())  # среднее
df['age'].fillna(df['age'].median())  # медиана
df['salary'].fillna(df['salary'].mode()[0])  # мода
```

**4. Заполнить по группам**
```python
df['age'].fillna(df.groupby('gender')['age'].transform('mean'))
```

**5. Использовать модель (MICE, KNNImputer)**
```python
from sklearn.impute import KNNImputer
imputer = KNNImputer(n_neighbors=3)
df_filled = imputer.fit_transform(df)
```

### Частая ошибка:
Удалять ВСЕ строки с пропусками — теряешь данные. Или заполнять средним там, где нужна медиана (выбросы!).
