# ML. Урок 47/3 — Сбор данных
## API

API — способ забирать данные напрямую из внешних сервисов: соцсети, погода, гео, финансы, курсы валют.

**Плюсы:** данные актуальные, можно автоматизировать сбор, легко обновлять.

**Минусы:** ограничения по запросам (rate limits), нужны ключи, иногда платно.

### Примеры
- OpenWeatherMap: погода по координатам
- Google Maps API: гео-данные, расстояния
- ЦБ РФ: курсы валют
- Dadata: ФИО, адреса, организации
- API ваших внутренних систем и открытых госданных

### Практический совет
Качай с запасом. Rate limit может сбросить сессию. Ставь retry с задержкой. И обязательно логируй ошибки — API может поменяться без предупреждения.

### Типичная ошибка
Написал скрипт, собрал 10 тысяч записей, на следующий день API изменило формат ответа. Скрипт упал, данные не обновились. Добавь обработку ошибок и мониторинг с первого дня.
