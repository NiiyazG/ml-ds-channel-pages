# Урок 46/3. Нормировка (Min-Max): когда данные в известных границах

### RU
Нормировка (Min-Max Scaling) — это сжатие всех значений признака в диапазон **[0, 1]** (или [-1, 1]).

**Формула:**

```
x_scaled = (x - min) / (max - min)
```

**Что происходит:**
- Самое маленькое значение → 0
- Самое большое значение → 1
- Все остальные → пропорционально между 0 и 1

**Пример:**
Цены квартир в городе:
- Минимальная: 2 000 000 ₽
- Максимальная: 20 000 000 ₽
- Квартира за 8 000 000 ₽ → (8 - 2) / (20 - 2) = 6/18 = 0.33

**Когда Min-Max лучше, чем StandardScaler:**
✅ Данные равномерно распределены (нет колокола)
✅ Признак ограничен известными границами (возраст 0-120, температура -50..+50)
✅ Нужны строго неотрицательные значения (например, для нейросетей с ReLU)
✅ Есть выбросы, но их можно отсечь (например, 1-99 перцентили)

**Когда Min-Max плох:**
❌ Есть сильные выбросы — они сожмут весь диапазон в кучку
❌ Новые данные выходят за границы обученного min/max

**Что делать с выбросами при Min-Max:**
Вместо честных min и max используют 1-й и 99-й перцентили:

```python
import numpy as np
from sklearn.preprocessing import MinMaxScaler

# Вариант 1: стандартный
scaler = MinMaxScaler()
scaled = scaler.fit_transform(data)

# Вариант 2: с отсечением выбросов
p1, p99 = np.percentile(data, [1, 99])
data_clipped = np.clip(data, p1, p99)
scaler = MinMaxScaler()
scaled = scaler.fit_transform(data_clipped)
```

**Сравнение с StandardScaler:**

| Характеристика | MinMaxScaler | StandardScaler |
|---|---|---|
| Диапазон | [0, 1] | Нет границ |
| Устойчив к выбросам | ❌ | Лучше, но не идеально |
| Сохраняет распределение | Линейно | Да |
| Для нейросетей | ✅ | ✅ |
| Для KNN/расстояний | ✅ | ✅ |

**Короткое правило:** Если знаешь границы данных — Min-Max. Если данные «колоколом» — StandardScaler.

---
### EN
Min-Max scaling compresses values to [0, 1] range. Best when you know the data boundaries and don't have strong outliers. Sensitive to outliers — they compress everything else into a tiny range. Use percentile clipping to handle outliers:

```python
scaler = MinMaxScaler()
scaled = scaler.fit_transform(data)
```
